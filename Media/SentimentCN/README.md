# 舆情洞察解决方案
舆情洞察解决方案是针对媒体行业的需求，利用微软自然语言处理技术、Azure云服务及大数据分析技术构建的行业解决方案。
解决方案利用互联网公开新闻文本数据，分析全球各主要国家及地区的热点新闻事件，持续监控事件趋势，多维度洞察事件影响力，协助媒体机构进行素材采集及选题。

## 场景描述
基于采集的互联网新闻数据，利用文本聚类算法、文本情感分析算法等自然语言处理技术发现热点事件，分析热点事件情感走势，及热点事件的影响力。
